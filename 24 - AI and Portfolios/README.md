# Portfolio Day: Guiding AI Through EDA

This folder contains everything you need for today's session. Read this page first, then follow the instructions during the activity.

## What You Will Do Today

1. **Watch a live demo** of structured vs. unstructured AI-driven EDA. Notice the difference in output quality.
2. **Write an EDA spec** with your group (3-4 people, ~10 minutes). Pick one of the two business scenarios, then fill in the spec template together.
3. **Feed your spec to an AI agent** and run a real analysis on the synthetic dataset provided.
4. **Challenge the results** using Pass 2 prompts. Push the AI to find alternative explanations and risks you missed.

## During the Activity

1. Pick Scenario 1 or Scenario 2 with your group.
2. Read the scenario card (2 minutes).
3. Open `spec-template.md` and fill in all five sections together (8 minutes). Start with what you know, then ask the AI to fill gaps.
4. Open your AI tool of choice (see below) and paste your completed spec as the starting prompt, along with the dataset.
5. When results come back, switch to Challenge Mode. See the Pass 2 section in `spec-example.md` for example prompts.

## Recommended AI Coding Tools

You do not need all of these. Pick one and get comfortable with it. All three have free access for students.

### Cursor

A VS Code fork with AI built into every workflow. Code generation, inline chat, multi-file edits, and agent mode for autonomous task completion. This is the most popular AI-native editor right now.

- **Free for students:** One full year of Cursor Pro (normally $20/month). Verify with your .edu email or GitHub Student Developer Pack.
- **Sign up:** [cursor.com/students](https://cursor.com/students)
- **Without student verification:** The free Hobby plan gives you 2,000 completions and 50 slow premium requests per month.

### VS Code + GitHub Copilot

The standard VS Code editor with GitHub Copilot for AI code completion and chat. If you already use VS Code, this is the lowest-friction option.

- **Free for students:** Unlimited Copilot access through GitHub Education. Verify your student status to unlock the full Student plan.
- **Sign up:** [education.github.com/pack](https://education.github.com/pack)
- **Without student verification:** Copilot Free gives everyone 2,000 completions and 50 chat messages per month, no credit card required. Just sign into VS Code with your GitHub account.

### Google Antigravity (Gemini)

Google's agent-first IDE, built on a VS Code fork and powered by Gemini models. Designed for delegating entire tasks to autonomous AI agents. Currently in public preview.

- **Free for everyone:** Antigravity is free during public preview. Sign in with a personal Gmail account.
- **Get started:** [codelabs.developers.google.com/getting-started-google-antigravity](https://codelabs.developers.google.com/getting-started-google-antigravity)
- **Note:** Still in preview as of early 2026. Expect occasional instability.

### Claude (for non-coding AI work)

For brainstorming, spec writing, challenging your analysis, and drafting documentation, Claude is excellent even without a coding IDE.

- **Free tier:** Sign up at [claude.ai](https://claude.ai) with any email. No credit card required.
- **Free for students at partner universities:** Sign up with your .edu email to check if BYU is a partner institution. If so, you get premium access automatically.
- **Education program:** [claude.com/solutions/education](https://claude.com/solutions/education)

## After the Session

Use the `reference/portfolio-framework.md` to evaluate your own portfolio projects against the three-tier framework. The self-evaluation checklist and README template are designed to be used directly. The `reference/resources.md` file has a curated reading list to go deeper.

The key takeaway: structure your AI use with the two-pass workflow (Prep Mode, then Challenge Mode), and always apply the three tests before you ship anything. Close the chat and explain it. Teach someone. Defend it tomorrow.
